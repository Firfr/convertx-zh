# 本地调试命令记录

当前项目翻译得到不是原码，是从docker镜像里面提取的编译后的前端文件。

## 获取前端代码

- 删除旧的前端代码
  ```bash
  rm -rf ./代码/*
  ```
- 拉取镜像
  ```bash
  docker pull c4illin/convertx:v0.15.1
  ```
- 运行命令
  ```bash
  docker run --rm -it \
    --name convertx \
    -v ./代码:/code \
    --entrypoint="" \
    c4illin/convertx:v0.15.1 \
    bash
  ```
  ```bash
  docker run --rm \
    --name convertx \
    -v ./代码:/code \
    --entrypoint sh \
    c4illin/convertx:v0.15.1 \
    -c '
      cp -rf /app/dist /code &&
      cp -rf /app/public /code &&
      echo "✅ 文件已复制并覆盖，容器即将退出"
    '
  ```

## 翻译调试

```bash
docker run -itd \
  --name convertx \
  -e JWT_SECRET=c4illin_convertx \
  -e HTTP_ALLOWED=true \
  -e LANGUAGE=zh \
  -e ACCOUNT_REGISTRATION=true \
  -v ./代码/dist:/app/dist \
  -v ./代码/public:/app/public \
  -v ./ts-调试/data:/app/data \
```
